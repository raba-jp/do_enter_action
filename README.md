# do_enter_action

This script displays current file list, git status and my last commit.

## Installation
### fisherman
```
fisher raba-jp/do_enter_action
```

## Example
```
function fish_user_key_bindings
  # Other key bindings
  bind \r enter_action
end
```

## Optional
If `exa` installed, this plugin uses it.
